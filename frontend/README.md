# ModuPlan Frontend

React + JavaScript + Vite 기반 프론트엔드입니다.

## Stack

- React
- React Router
- React Query
- React Hook Form
- Axios
- Vite
- Tailwind CSS

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Structure

```text
src/
  api/
  components/
  constants/
  hooks/
  layouts/
  pages/
  routes/
  services/
  store/
  utils/
```

## Notes

- 현재 소스 코드는 `js/jsx` 기준으로 정리되어 있습니다.
